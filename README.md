# Exercism Discussions

This repository is for all the discussions where the action is not yet clear, where a decision needs making, a feature needs designing, or a policy needs setting. This is the repository for philosophical discussions, for asking clarifying questions, for discussing priorities.

The actual discussions can be found in the [issues](https://github.com/exercism/discussions/issues) section.

If you would like to start a discussion, please open your issue at [exercism/exercism](https://github.com/exercism/exercism) and it will be triaged here if appropriate. Thank you :blue_heart:
